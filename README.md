# Feliz 7mo Aniversario

Una experiencia romántica e interactiva creada para celebrar siete meses juntos.

## Incluye

- Siete capítulos con recuerdos y fotografías reales.
- Álbumes adaptados para celular.
- Paisaje sonoro generado en el navegador.
- Ramo interactivo de siete tulipanes.
- Reproductor oficial de Cantarina.

## Despliegue

El proyecto es una web estática y puede desplegarse directamente en Vercel sin comando de compilación.
